# Project Summary: Heart Disease Risk Prediction

## Problem

Coronary heart disease is a major health risk. This project uses patient-level health indicators to predict whether an individual may develop CHD within 10 years.

## Data

The dataset contains 4,238 observations and 16 columns. The target variable is `TenYearCHD`.

## Approach

1. Load and inspect the dataset
2. Check missing values
3. Impute missing numeric values
4. Split features and target
5. Train multiple classification models
6. Evaluate models using cross-validation

## Models

- Logistic Regression
- Naive Bayes
- Decision Tree
- Random Forest
- Support Vector Machine

## Important Note

This is a data science portfolio project and should not be used for real medical diagnosis.
